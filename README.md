
# Luxury Dermatology Website

A modern, fully responsive dermatology website built with HTML, CSS, and JavaScript. This project demonstrates my ability to create premium, conversion-focused user interfaces with clean code, responsive layouts, and polished user experiences.

## 🌐 Live Demo

https://starlit-queijadas-199702.netlify.app/

---

## 📌 Project Overview

This project was designed as a luxury website for dermatology and aesthetic clinics. The primary goal was to create a modern digital experience that combines elegant visual design with responsive front-end development.

The website emphasizes usability, accessibility, and performance while maintaining a premium aesthetic suitable for high-end medical and skincare brands.
